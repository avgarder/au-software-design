Основной цикл в методе main файла Main.kt.
Класс Shell предоставляет интерфейс для взаимодействия с внешним миром и переменными окружения.
Класс CommandBuilder конвертирует введённый скрипт в набор команд, последовательно вызывая вспомогательные методы других классов.
Класс Preprocessor отвечает за замену переменных окружения в скрипте.
Класс Tokenizer разбивает скрипт на отдельные "слова".
Класс PipeSplitter разбивает последовательность слов на последовательность команд.
Класс CommandFactory преобразует строковое представление команды в класс, реализующий команду.
Все команды имеют общий интерфейс для создания и запуска.
